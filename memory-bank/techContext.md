# Teknik Bağlam

## Kullanılan Teknolojiler
- **Runtime/Build**: Node.js, Vite.
- **Diller**: HTML5, Modern JavaScript (ES6+), Vanilla CSS.
- **Kütüphaneler**: GSAP (Animasyon), ScrollTrigger.
- **Tipografi**: Google Fonts (Epilogue, Inter, Montserrat - Tasarım sistemine göre seçilecek).

## Geliştirme Kurulumu
- `npm install` ile bağımlılıkların yüklenmesi.
- `npm run dev` ile yerel sunucunun başlatılması.
- Tasarım sistemi değişkenlerinin `index.css` içinde tanımlanması.

## Teknik Kısıtlamalar
- **Vanilla CSS**: Tailwind veya benzeri kütüphaneler (istek gelmedikçe) kullanılmayacak.
- **Performans**: Animasyonlar cihaz performansını düşürmemeli (GSAP context kullanımı).
- **SEO**: Her sayfa için meta tag'ler ve doğru başlık hiyerarşisi.

## Bağımlılıklar
- Vite (Build tool)
- GSAP (Animation library)
- (Opsiyonel) Handlebars (Static site generation için)
